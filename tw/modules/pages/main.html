Server PCB

各式 Server 平台生產經驗豐富，針對各種 Server PCB 所需要的材料與技術要求，非常完備。 無論是高頻低損耗材料選型、高層數疊構設計，或是 Back-drill 工藝，博智皆具備完整的量產能力。

高層數 Back-drill 低損耗材料 高速訊號 大板尺寸

Networking PCB

100G、400G、800G Switch PCB 生產經驗豐富，此類 PCB 對材料損耗的要求高， 博智有完整的電性實驗室，對材料電性的掌握優於同業，可確保訊號完整性達標。

100G Switch 400G Switch 800G Switch 低介電損耗 電性驗證
洽詢 Networking PCB 規格：sales@accl.com.tw
IPC 工業電腦 PCB

各式嵌入式電腦方案 PCB 主要供應商，少量多樣是此類 PCB 的特性， 生產時換料頻繁，精實的製造管理是重點。博智以彈性生產及嚴格的變更管制確保品質穩定。

嵌入式系統 少量多樣 精實生產 快速換線
洽詢 IPC PCB 規格：sales@accl.com.tw
IC 測試板

可生產 Probe Card、Load board、測試設備所需的 PCB，此類 PCB 需要整合 VIPPO、Backdrill、Laser Via、Sequential Stack 等工藝於同一片板上， 博智對此混合式生產經驗豐富，是業界少數具備量產能力的廠商。

Probe Card Load Board VIPPO Backdrill Sequential Stack Laser Via
洽詢 IC 測試板規格：sales@accl.com.tw
